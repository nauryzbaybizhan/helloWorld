1) Чтобы скачать проект, открываем целевую папку для клонирования проекта из репозитория github и вводим в git bash команду:
git clone https://github.com/nauryzbaybizhan/helloWorld.git
2) Открываем папку с проектом в командной строке, для запуска сервера вводим команду:
java -jar helloWorld-0.0.1-SNAPSHOT.jar
